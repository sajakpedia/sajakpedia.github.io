1. Sepasang sepatu + sepasang sepatu + sepasang sepatu = 60Sepasang sepatu + 1orang +1orang = 301Kacamata + 1orang + 1kacamata = 9Sepasang sepatu + sebelah sarung tangan + 1kacamata = 42Sebelah sepatu + satu orang berkacamata bersarung tangan bersepatu × 1kacamata=?​


Jawaban:

dimisalkan : • sepasang sepatu = x

• 1 orang = y

• 1 kacamata = z

• sarung tangan = a

Diket : • x + x + x = 60

• x + y + y = 30

• z + y + z = 9

• x + ½a + z = 42

Ditanya : ½x + (y + z + a + x)z

Jawab : • x + x + x = 60

3x = 60

x = 60/3

x = 20

• x + y + y = 30

20 + 2y = 30

2y = 30 - 20

2y = 10

y = 10/2

y = 5

• z + y + z = 9

2z + 5 = 9

2z = 9 - 5

2z = 4

z = 4/2

z = 2

• x + ½a + z = 42

20 + ½a + 2 = 42

22 + ½a = 42

½a = 42 - 22

½a = 20

a = 20/½

a = 40

Maka, ½x + (y + z + a + x)z

= ½ . 20 + (5 + 2 + 40 + 20)2

= 10 + 67 . 2

= 10 + 134

= 144

Jadi, sebelah sepatu + satu orang berkacamata bersarung tangan bersepatu × 1kacamata adalah 144

p + 3q = 140.000,00................(ii)

p(4q) + 3q = 140.000

⇒ 7q = 140.000 ⇒ q = 20.000

maka, nilai dari p = 4 ( 20.000 ) = 80.000

jadi, harga sepasang sepatu dan sepasang sandal

p = 80.000

q = 20.000

klo mau beli dua - duanya tinggal ditambahin

p + q

80.000 + 20.000 = 100.000,00
