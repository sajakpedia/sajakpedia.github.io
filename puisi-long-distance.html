4. Tolong terangkan mengenai dengan long distance learningdan e-Learning.


long distance learning dan e-Learning merupakan bentuk pengimplementasian yang baik dan benar dari teknologi informasi dan komunikasi atau TIK. Long distance learning merupakan model pembelajaran jarak jauh sehingga antara pengajar dan siswanya tidak perlu berada di satu tempat yang sama. Hal ini memberikan keuntungan jika seorang siswa tidak memungkinkan ke tempat belajar dimana pengajar tersebut mengajar, namun tetap bisa melakukan kegiatan belajar mengajar. Penerapan long distance learning bisanya menggunakan media elektronik seperti internet, sehingga disebut dengan e-learning. Penggunaan jaringan internet inilah yang memungkinkan seseorang dalam melakukan pembelajaran jarak jauh. Internet bisa menghubungkan suatu komputer dengan suatu jaringan atau komputer lain meskipun berjauhan jaraknya. Media internet inilah menjadi jembatan bagi pengajar untuk mengirimkan materi pembelajarannya kepada siswanya untuk belajar.

Rumus negatif : S+ (are,am,is) + not + object

S + don't/doesn't + verb1 + O

rumus kalimat tanya = Do/does + S + verb1 + O

(are,is,am) + S + verb1 + O

Note : are untuk you,they,we

am untuk i

is untuk she,he,it

does untuk she,he,it

do untuk i,you,they,we
