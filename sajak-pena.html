15. Kalau pensil 2 dan pena 3 : 8000dan pensil3 dan pena 2:7000berapa pena 1


Misal pensil (x) dan pena (y)

2x + 3y = 8.000 | × 2

3x + 2y = 7.000 | × 3

-----------------------

4x + 6y = 16.000

9x + 6y = 21.000

----------------------- -

- 5x = - 5.000

x = 1.000 ( Pensil )

ubah x ke salah satu persamaan.

3x + 2y = 7.000

3(1.000) + 2y = 7.000

3.000 + 2y = 7.000

2y = 7.000 - 3.000

2y = 4.000

y = 4.000/2

y = 2.000 ( Pena )

Jadi, harga 1 pena adalah 2.000
