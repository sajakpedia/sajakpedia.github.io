Penyelesaian ( 1 )

12² + 10²

= ( 12 x 12 ) + ( 10 x 10 )

= 144 + 100

= 244 ✔

Penyelesaian ( 2 )

15² - 11²

= ( 15 x 15 ) - ( 11 x 11 )

= 225 - 121

= 104 ✔
