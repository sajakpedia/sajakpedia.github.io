kakek agung = A
pak Budi = B
nenek CINTA = C

diket :
A = 56+B.........(1)

C = A-12
A =12+C...........(2)

A + B + C = 238........(3)

persamaan 1 & 2 → jumlahkan

A = 56+B

—————+
