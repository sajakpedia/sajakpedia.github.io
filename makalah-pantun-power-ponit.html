1. Cara mebuka MS. power ponit melaui shotrtcut pada desktop adalah


Jawaban:

1. Klik ikon Microsoft PowerPoint pada desktop Anda.

2. Tekan tombol Windows + R pada keyboard Anda untuk membuka jendela Run.

3. Ketik "powerpoint" di jendela Run dan tekan Enter.

4. Jika Anda menggunakan versi Office yang lebih baru, Anda dapat mengetikkan "powerpoint" di jendela Run dan tekan Enter.

2. Sebutkan masing-masing fungsi dari 6 bagian elemen dasar jendela / menu pada Microsoft Power Ponit!


Tab Home pada Ms. powerpoint 2007 memiliki 6 bagian elemen yang mana masing-masing elemen disebut grup dan mempunyai fungsi-fungsi tertentu:
1. Grup Clipboard: memiliki icon cut (untuk memotong objek/ teks yang diblok/ disorot), memiliki icon copy (untuk menyalin objek/ teks yang diblok), memiliki icon paste (untuk menempel objek/ teks yang diblok).

2. Grup Slides: memiliki icon New Slide (untuk membuat slide baru), memiliki icon Layout (untuk mengatur layout slide yang aktif), memiliki icon Delete (untuk menghapus slide yang terpilih.

3. Grup Fonts: memiliki icon-icon untuk mengatur jenis teks, ukuran teks, efek teks.

4. Grup Paragraph: memiliki icon-icon untuk mengatur perataan paragraf, jarak spasi, mengatur penomoran paragraf dan lainnya.

5. Grup Drawing: memiliki icon-icon untuk menyisipkan objek ke dalam slide powerpoin serta memberikan pengaturan efek pada objek tersebut

6. Grup Editing: memiliki icon-icon yang berfungsi untuk mencari teks (Find), mengganti teks (Replace) dan memilih objek (Select)
