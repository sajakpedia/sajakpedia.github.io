Jawaban:

x + y + z = 349 -----------------------> ( 1 )

x = miko

y = keddy

z = sharko

x = 3y ---------------> y = 1/3 x -----> ( 2 )

z = x + 202 ---------------------------> ( 3 )

ditanya x......???

selesaiannya :

x + y + z = 349

x + ( 1/3 x ) + ( x + 202 ) = 349

7/3 x = 349 - 202

7/3 x = 147

x = 147 × 3/7

x = 63

jadi perangko yg dimiliki miko berjumlah

63 buah.

25. Miko, Keddy dan Sharko mempunyai total 438 buah prangko. Miko mempunyai prangko 3 kali lebih banyak daripada Keddy. Sharko mempunyai 263 prangko lebih banyak daripada Miko. Berapakah jumlah prangko yang dimiliki oleh Miko?


Jawaban:

75 prangko

Penjelasan dengan langkah-langkah:

Misalkan

Miko = m

Keddy = k

Sharko = s

m + k + s = 438

3k + k + 263 + m = 438

4k + 263 + 3k = 438

7k + 263 = 438

7k = 438 - 263

7k = 175

k = 175 : 7

k = 25 prangko

m = 3k

= 3 × 25

= 75 prangko

Jadi, jumlah prangko yang dimiliki oleh Miko adalah 75 prangko

Sn=a×(r^n - 1)/(r-1) untuk r>1 atau Sn=a×(1-r^n)/(1-r) untuk r<1] Karena ada 6 anak, jadi barisan geometrinya terdiri dari 6 suku. Kita misalkan anak pertama mendapat a rupiah, anak kedua mendapat a×r^2-1=ar¹=ar, dan seterusnya sehingga barisannya menjadi: a, ar¹, ar², ar³, ar⁴, ar⁵ Suku ke 2=240.000 ar=240.000 a=240.000/r.......(persamaan 1) Suku ke 5 = 30.000 ar⁴=30.000 a=30.000/r⁴......(persamaan 2) Dari persamaan 1 dan 2 kita peroleh: 240.000/r=30.000/r⁴ Dengan perkalian silang, kita peroleh: 240.000×r⁴=30.000×r Kita kumpulkan yang ada variabel r di ruas kiri, menjadi: r⁴/r=30.000/240.000 r³=⅛ r=½ Karena r=½, maka dapat kita cari nilai a menggunakan persamaan 1 atau 2 a=240.000/r a=240.000/(½) a=240.000×2 a=480.000 Karena r<1, menggunakan: Sn=a×(1- r^n)/(1-r) Sn=480.000×(1-½⁶)/(1-½) Sn=480.000×(1- 1/64)/(½) Sn=480.000×(63/64)×2 Sn=945.000 Jadi, banyak uang yang dibagikan Pak Miko setiap minggunya adalah Rp945.000,00.
