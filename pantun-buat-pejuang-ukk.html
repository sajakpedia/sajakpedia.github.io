13. Bantuin dong, buat UKK nanti.


diketahui:
AB =DC = HG = EF = 9 cm
BC = AD = EH = FG = 12 cm
AE = FB = GC = HD = 20 cm

ditanya:
keliling ∆ ACG =?

dijawab:
AB² + BC² = AC²
9² + 12² = AC²
81 + 144 = AC²
AC² = 225
AC = √225
AC = 15 cm

AC² + GC² = AG²
15² + 20² = AG²
225 + 400 = AG²
AG² = 625
AG= √625
AG = 25 cm

keliling ∆ ACG = AC + AG + GC = 15 + 25 + 20 = 60 cm

jadi, keliling ∆ ACG adalah 60 cm (A)

39. tolong dong ... buat belajar ukk


5x + 6° + 3x + 4° = 90°     (saling berpenyiku)
              8x + 10° = 90°
                        8x = 90° - 10°
                        8x = 80°
                          x = 80° / 8
                          x = 10°

∠ BOC = 5x + 6°
             = 5 (10°) + 6°
             = 50° + 6°
             = 56°    ....... jawaban : Ddiket: <boc=(5x+6)°
<boa=(3x+4)°
<aoc=90°(merupakan sudut suku-suku)
ditanya: <...?
jawab:
rumus: <aoc=<boc+<boa
90°=(5x+6)+(3x+4)
90°=5x+3x+6+4
90°=8x+10
90-10=8x
80=8x
x=80/8
x=10°
jadi <boc=(5x+6)
=5(10)+6
=50+6
=56°
